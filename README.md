# Age Calculator

A simple beginner Python project that calculates a person's age based on their birth year.

## Features
- Calculates age
- Classifies age group (minor, adult, senior)
- Simple and beginner-friendly

## How to Run

1. Make sure Python is installed.
2. Run the following command:

```bash
python age_calculator.py
```

## Example

Enter your birth year: 2005  
Your age is: 20 years  
You are an adult.
